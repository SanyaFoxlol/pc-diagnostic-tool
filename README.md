# Python System Monitor

Simple real-time system monitoring tool written in Python.

This project displays basic system information and updates it live in the terminal.

## Features

* Real-time CPU usage monitoring
* RAM usage monitoring
* Disk usage monitoring
* Hostname and IP address display
* Console progress bars
* Screen refresh without printing new lines

## Technologies

* Python 3
* psutil library

## Installation

1. Install Python 3
2. Install required library:

```
pip install psutil
```

## Usage

Run the program:

```
python monitor.py
```
